# ケータリング在庫表

ケータリング用品の在庫を確認し、スクリーンショット共有しやすくするための単一HTMLツールです。

## 公開URL

- https://taka0326y.github.io/cafe-stock/

## 主な機能

- 在庫数の確認
- 注意が必要な項目の把握
- 日付・担当者情報の入力
- スマホ表示対応
- GitHub Pagesでの共有

## 主なファイル

| ファイル | 内容 |
|---|---|
| `index.html` | 在庫表本体 |
| `README.md` | この説明ファイル |

## 運用メモ

既存の共有URLを維持するため、リポジトリ名と公開ファイル名は変更しません。現場で使いやすいように、必要な項目は `index.html` に直接反映します。

## 利用範囲

個人利用・共有用のページです。再利用したい場合は作者に確認してください。
